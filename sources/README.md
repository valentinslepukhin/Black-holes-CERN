# Sources

Primary sources used in this audit. PDFs are **not** committed (copyright); fetch
them from the canonical links below.

- **Giddings & Mangano (2008)**, *Astrophysical implications of hypothetical
  stable TeV-scale black holes*, CERN-PH-TH/2008-025.
  arXiv: [`0806.3381`](https://arxiv.org/abs/0806.3381) — Phys. Rev. D 78, 035009.
  The load-bearing reference for this project. Pages cited throughout `notes/`:
  Table 1 (p.36), §§4.5–4.6 (pp.27–28), §5.3.2 + Figs 1–2 (pp.35–37),
  §§7.4–8.1 (pp.44–45), §9 conclusions (pp.50–53), App. F + Tables 11–12 (pp.79–83).

To work locally with the PDF:

```bash
curl -L -o sources/giddings-mangano-0806.3381.pdf https://arxiv.org/pdf/0806.3381v2
```
